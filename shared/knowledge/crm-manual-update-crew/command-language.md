# CRM Manual Update Crew Command Language

## Quick commands

- `skip`
- `nota: Texto de la nota`
- `mover a Proposal`
- `importe 16000`
- `crear tarea llamar a Josefa el viernes`
- `cerrar tarea enviar dossier`
- `siguiente paso: enviar propuesta revisada`
- `resumen`
- `confirmar`
- `cancelar`

## Natural language

Free-form updates are allowed. The crew may turn them into note material only
when the facts are explicitly provided by the user.

## Ambiguity

If a command can match multiple stages, tasks, deals, companies, or contacts,
the crew asks the user to choose.
