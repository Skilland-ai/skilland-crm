# CRM Execution Crew Skills

Repo-local skill specs for the CRM Execution Crew. Runtime implementations live
under `scripts/crm_execution_crew/skills/`.

## Skill set

- `twenty-docs-search`
- `twenty-metadata`
- `twenty-record-search`
- `crm-plan-validation`
- `crm-execution`

## Rule

Skills are tools. They do not decide global strategy; the Orchestrator Agent and
Safety Reviewer Agent compose and gate their outputs.
