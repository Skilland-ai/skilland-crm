# Twenty Metadata

## Purpose

Use `/rest/metadata/objects` to validate CRM operation requests.

## Outputs

- objects checked
- fields by object
- unknown fields
- invalid select options
- warnings
- blocking issues

## Restrictions

- metadata read only
- no metadata mutations
- no CRM record writes
