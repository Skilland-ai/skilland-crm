# Twenty Docs Search

## Purpose

Find source-backed local docs under `packages/twenty-docs/`.

## Outputs

- docs consulted
- reason for each path
- findings
- implications
- warnings

## Restrictions

- no web search
- no CRM reads or writes
- no invented documentation
