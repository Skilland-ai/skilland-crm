# IA Mujeres CRM/GWS — How To Use

Fecha: 2026-06-08

## Centro operativo

Twenty CRM es el centro de mando. Gmail es canal de envío y señal, pero el estado comercial vive en:

- vista `IA Mujeres — Funnel`;
- campo `iaMujeresFunnelStage`;
- campos Gmail en Opportunity;
- notas;
- tareas.

## Operación diaria segura

```bash
node scripts/ia_mujeres_daily_operator.mjs --limit=5 --weekly
```

Este comando:

- audita CRM;
- selecciona siguiente tanda;
- genera payloads locales;
- revisa follow-ups;
- genera reporte semanal;
- no muta CRM;
- no crea Gmail drafts;
- no envía emails.

## Runner CRM

```bash
node scripts/ia_mujeres_batch_runner.mjs --mode=audit
node scripts/ia_mujeres_batch_runner.mjs --mode=select-batch --limit=5
node scripts/ia_mujeres_batch_runner.mjs --mode=prepare-drafts --batch-id=<id>
node scripts/ia_mujeres_batch_runner.mjs --mode=mark-draft-created --batch-id=<id> --draft-map=<json> --apply
node scripts/ia_mujeres_batch_runner.mjs --mode=mark-email-sent --batch-id=<id> --sent-map=<json> --apply
node scripts/ia_mujeres_batch_runner.mjs --mode=sync-replies --apply
node scripts/ia_mujeres_batch_runner.mjs --mode=sync-bounces --apply
node scripts/ia_mujeres_batch_runner.mjs --mode=prepare-followups --limit=5
```

## Reglas de autorización

- `send-approved` está bloqueado.
- Todo cambio CRM requiere `--apply`.
- Todo envío externo requiere aprobación humana explícita.
- No usar outputs antiguos como revisión vigente; usar batch `2026-06-08T00-34-36-009Z` o generar uno nuevo.

## Reporte semanal

```bash
node scripts/ia_mujeres_weekly_report.mjs --week=2026-06-08
```

Genera Markdown y HTML; no envía email ni muta CRM.
